    Host This Calculator Free on GitHub Pages

    GitHub Pages hosts static HTML files for free. Once set up, your calculator will be live at a public URL — no server, no cost.

    Step 1. Create a free account at github.com if you don't have one.
    Step 2. Click + New repository → name it something like rate-calculator → check Public → Create.
    Step 3. Inside the repo click Add file → Upload files → drag and drop the index.html file you downloaded.
    Step 4. Go to Settings → Pages → under Source select main branch → click Save.
    Step 5. Wait ~60 seconds → your calculator is live at:
    https://YOUR-USERNAME.github.io/rate-calculator/
    💡 To update later: go back to the repo → click index.html → pencil icon → paste updated code → Commit changes.
    Access Control Tips

    GitHub Pages public repos are visible to anyone with the URL. To restrict access:
    • Share the link only with your team (no login = security by obscurity)
    • For real auth: use GitHub Enterprise or add a password prompt in JavaScript
    • Consider Netlify or Cloudflare Pages if you need more control
